ARTEMIS II TRACKER

Real-time trajectory display for NASA's Artemis II mission — the first crewed flight to the Moon in over 50 years.

Orion's position is tracked using live telemetry from NASA's AROW system. The predicted trajectory comes from JPL Horizons.

This is a vibe-coded derivative work of Ian Dees' artemis-viewer.
